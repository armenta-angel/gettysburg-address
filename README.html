La kata consiste en dividir una frase (en este caso extraída del discurso de Gettysburg) en líneas de menos de 13 caracteres.

La kata está basada en el problema planteado por Uncle Bob en el post .

En el post se describe el problema de que las estimaciones no pueden basarse en la aparente simplicidad de como los humanos podemos resolver un problema sencillo. Resolver el problema en un ordenador requiere llevar dicha resolución a un sistema procedural y esta tarea resulta a veces complicada.
El post termina con una gran frase: "And if anyone tries to tell you that your estimate is bogus because the task is so simple, ask them to write down the procedure for tying their shoes."
